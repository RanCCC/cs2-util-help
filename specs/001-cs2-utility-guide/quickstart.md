# Quickstart: CS2 Utility Lineup Guide

**Branch**: `001-cs2-utility-guide` | **Date**: 2026-04-05

## Prerequisites

- **Node.js 20+** installed ([download](https://nodejs.org/))
- A modern web browser (Chrome, Firefox, or Edge)

## First-Time Setup

```bash
# Clone or download the project
cd cs2-util-help

# Install dependencies
npm install

# Build the frontend
npm run build

# Start the app
npm start
```

The app opens automatically at `http://localhost:3001`.

## For Non-Technical Users (Windows)

**Double-click `start.bat`** — that's it.

The batch file will:
1. Check that Node.js is installed (prompts to download if not)
2. Install dependencies if needed (`npm install`)
3. Build the frontend if needed (`npm run build`)
4. Start the server
5. Open the browser to `http://localhost:3001`

To stop the app, close the terminal window.

## Adding Map Images

Before using the app, add your CS2 map images:

1. Open the `data/` folder next to the app
2. Create a folder for each map (e.g., `dust2/`, `mirage/`)
3. Place a top-down map image named `map.png` (or `.jpg`/`.webp`) in each folder
4. Edit `data/maps.json` to register the maps:

```json
{
  "maps": [
    { "id": "dust2", "name": "Dust 2", "folder": "dust2", "image": "map.png" },
    { "id": "mirage", "name": "Mirage", "folder": "mirage", "image": "map.png" }
  ]
}
```

5. Restart the app (or refresh the browser)

## Development Mode

```bash

npm run dev:server

# In a second terminal, start the Vite dev server
npm run dev

# Frontend is at http://localhost:5173, proxying API calls to :3001
```

## Common Tasks

### View Lineups
1. Open the app → select a map
2. Landing points (where utilities land) are shown for Smoke by default
3. Click a landing point → see starting points connected by dotted lines
4. Hover a starting point → see lineup preview image
5. Click a starting point → opens detail page in a new tab

### Add a Lineup
1. On any map, switch to **Edit Mode**
2. Click the map to place a landing point
3. Click again to place a starting point
4. Upload the 3 required files: position screenshot, lineup screenshot, video
5. Enter a description and your name
6. Save

### Share Lineups
1. On any map, click **Export**
2. Select specific lineups or use "Select All"
3. Download the ZIP and send it (e.g., over Discord)
4. The recipient clicks **Import** and selects the ZIP file

## Troubleshooting

| Issue | Solution |
|-------|---------|
| "Node.js not found" | Install Node.js 20+ from https://nodejs.org/ |
| Port 3001 already in use | Close the other terminal running the app, or change PORT in `.env` |
| Map doesn't appear | Check that `data/maps.json` lists the map and the image file exists |
| Upload fails | Check file format (images: PNG/JPG/WebP, video: MP4/WebM) and that the `data/` folder is writable |
